About Kelowna Codes

At Kelowna Codes, we empower high school students to think and code like engineers. Our hands-on programs teach coding, problem-solving, and design skills in a fun, collaborative environment. Whether you're a beginner or already familiar with coding, we have the perfect program to help you build real-world projects and prepare for the future!

Our Programs

Every program includes deploying what you've built to the real world, and starting your own github code portfolio. Employers and undergrad applications love this!

Each program is available as a premade module, but we highly recommend tailoring them to the specific coder. Learn engineer-caliber skills while building something fun.

  • Web Site Development (Ages 13-18): Learn HTML, CSS, JavaScript, and build your first website.
  • Web App Development (Ages 14-18): Create your own apps using React, Javascript, and HTML. Choose from a preset selection of apps to create, or brainstorm your own idea!
  • Game Development (Ages 15-18): Build your own video games with Javascript, React and Node, and host them online!
  • Robotics & Engineering Challenges (Ages 16-18): FIRST tech challenge available by request.
